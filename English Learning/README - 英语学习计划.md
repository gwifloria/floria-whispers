# 英语学习计划

## 目标
重新捡回阅读、听力、口语能力（专八良好基础）

## 学习原则
1. **别完美主义** - 不要每篇文章都做笔记，读100篇只精读10篇
2. **建立习惯** - 每天30分钟，坚持3个月
3. **输出驱动输入** - 写英文日记、评论、笔记
4. **享受优先** - 选真正感兴趣的内容

## 能力重建路线

### 阅读（最容易捡回来）
- **每天读感兴趣的英文内容**
- **不要查每个词**：能猜出大意就继续
- **推荐来源**：
  - Hacker News 的讨论
  - The Atlantic, The New Yorker
  - Substack newsletter

### 听力（需要每天练）
- **播客**：通勤、做家务时听
  - Lex Fridman（深度访谈）
  - The Daily（新闻）
- **YouTube 1.25-1.5倍速**
- **关键**：听让你需要集中注意力但又能听懂70%的

### 口语（最难但最重要）
- **找语伴**：italki, HelloTalk
- **自言自语**：用英语描述你在做什么、想什么
- **跟读练习**：暂停后重复播客内容
