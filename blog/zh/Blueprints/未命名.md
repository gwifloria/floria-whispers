 First Glance 产品路线图 & 商业化方案

 Context

 First Glance 是一个 Chrome 新标签页扩展，当前版本 v1.3.5，用户量 <
 100。产品定位是"干净的效率工具"，不以盈利为最大化目标，但希望 AI
 功能的成本能被覆盖并有小额盈利。不希望自建后端，尽量用第三方服务。

 当前功能盘点（v1.3.x）

 免费功能（已完成）：
 - Focus/List 双视图
 - 滴答清单 / Todoist 集成 + 游客模式
 - 番茄钟（跨标签页同步、沉浸模式、任务绑定）
 - 网站屏蔽 + Chill Mode
 - AI Buddy（用户自带 API Key）
 - 5 种主题 + 中英文切换
 - Popup 快捷面板

 ---
 版本规划

## v1.4 — 体验打磨（免费，无需后端）

 目标：把核心体验做到位，让现有用户愿意推荐

 番茄钟提示音
 - 内置 3-4 个短音效（清脆铃声、木鱼、轻柔叮咚等）
 - work/break 结束时播放
 - 设置中可选择音效或关闭
 - 文件：src/assets/sounds/、usePomodoro.ts、设置面板

 专注数据统计（基础版，本地存储）
 - 记录每次番茄完成：时间戳、时长、绑定任务
 - 存储在 chrome.storage.local（无需后端）
 - 今日/本周的简单统计卡片：专注次数、总时长、完成任务数
 - 文件：新增
 src/services/focusStats.ts、src/components/FocusView/StatsCard.tsx

 白噪音（基础版）
 - 内置 2-3 种环境音（雨声、咖啡馆、纯白噪音）
 - 番茄钟启动时自动播放，结束时停止
 - 音量控制滑块
 - 音频文件可用免费素材（freesound.org CC0）
 - 文件：新增 src/hooks/useAmbientSound.ts、设置面板扩展

## v1.5 — Pro 体系搭建（引入付费，轻量后端）

 目标：建立 Free/Pro 分层，开始有收入

 技术基础设施
 - 支付：LemonSqueezy（专为独立开发者设计）
   - 支持订阅 + 一次性购买
   - 内置税务处理（全球合规）
   - Webhook 通知支付状态
   - 无需自建支付后端
 - 用户身份：复用现有的滴答清单/Todoist OAuth 登录
   - 用 OAuth 的 user ID 作为唯一标识
   - LemonSqueezy 的 custom_data 存 user ID，购买时关联
 - 授权验证：
   - LemonSqueezy License API 验证订阅状态
   - 扩展侧定期检查 license 有效性，结果缓存在 chrome.storage.local
   - 无需自建用户数据库

 定价方案
 Free（永久免费）                    Pro（$3/月 或 $24/年）
 ─────────────────                  ─────────────────────
 番茄钟全功能                        ✅ 全部 Free 功能
 基础任务管理                        AI Buddy（Pro 托管 API，无需自带 Key）
 网站屏蔽 + Chill Mode               专注数据统计（周报/月报/趋势图）
 5 种内置主题                        高级主题 + 自定义配色
 基础提示音（1 种）                   全部提示音 + 白噪音
 基础统计（今日）                     高级屏蔽规则（时间段、番茄钟联动）

 Pro 功能实现
 - AI Buddy Pro：
   - Free 用户：仍可自带 API Key 使用（保持现状）
   - Pro 用户：通过 Cloudflare Worker 代理 API 调用（无需自建服务器）
   - Worker 验证 LemonSqueezy license → 转发请求到 OpenAI/Claude
   - 每月限额（如 500 次对话），防止滥用
 - 高级统计：
   - 周报/月报视图（柱状图/趋势线）
   - 任务完成率分析
   - 数据仍存 chrome.storage.local（隐私友好）
 - 高级主题：
   - 新增 3-5 个 Pro 主题
   - 自定义配色（主色、背景色、强调色）
 - 全部提示音 + 白噪音：
   - Free 保留 1 个基础提示音
   - Pro 解锁全部提示音 + 白噪音
 - 高级屏蔽：
   - 按时间段自动屏蔽（如工作时间 9-18 点）
   - 番茄钟启动时自动激活屏蔽

 Pro 状态管理
 新增文件：
   src/services/license.ts          — LemonSqueezy license 验证
   src/contexts/ProContext.tsx       — Pro 状态 context
   src/components/common/ProBadge   — Pro 标识 & 升级引导

 Cloudflare Worker（独立仓库）：
   ai-proxy/                        — AI 请求代理
     src/index.ts                   — 验证 license + 转发请求

 - LemonSqueezy：支付、订阅管理、License 发放
 - Cloudflare Worker：AI 请求代理（免费额度 10 万次/天，足够早期）
 - chrome.storage：所有用户数据本地存储（无服务端数据库）
 - 无用户数据库：OAuth user ID + LemonSqueezy license key 做关联

 收支平衡估算

 成本（月）
 - Cloudflare Worker：免费（10 万次/天）
 - AI API：假设 Pro 用户 50 人，每人 500 次/月对话
   - GPT-4o-mini：~$0.15/千次 → 50 × 500 / 1000 × $0.15 = $3.75/月
   - Claude Haiku：类似量级
 - LemonSqueezy：5% 手续费
 - 域名/网站：~$10/年

 收入（月）
 - 50 个 Pro 用户 × $3/月 = $150/月
 - 扣手续费 5%：$142.5/月
 - 净收入：~$138/月

 → 50 个付费用户就能覆盖成本并有合理盈利

   ├── 番茄钟提示音

   └── 基础白噪音

 v1.5（之后 3-4 周）    → 引入 Pro

   ├── Pro 状态管理
   ├── AI Buddy Pro（CF Worker）
   ├── 高级统计
   ├── 高级主题
   └── 全部音效解锁

 v1.6（后续）           → 深度功能
   ├── 专注报告导出
   ├── AI 任务规划
   ├── 番茄钟模板
   └── 快捷键 / Widget

 关键决策点

 1. AI Buddy 双轨制：Free 自带 Key + Pro 托管 API，两种模式共存
 2. 无后端用户管理：OAuth ID + LemonSqueezy License，不建数据库
 3. 渐进式 Pro：v1.4 先做好免费体验，v1.5 再引入付费，避免功能不够就收费
 4. 隐私友好：统计数据全部本地存储，不上传服务器
